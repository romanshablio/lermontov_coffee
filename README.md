# lermontov_coffee

A promotional feedback website for a coffee shop.

The page lets visitors generate a unique promo code, submit contact information, rate the barista and product quality, and leave a review. The submitted form is handled by PHP scripts. The site also includes PWA-related assets such as a manifest and service worker.

## Features

- Promo code generation in the browser.
- Customer feedback form.
- Rating fields for friendliness, service speed, product quality, and cleanliness.
- Personal data policy page.
- PHP form handling.
- PWA manifest and service worker files.

## Technologies

- HTML
- CSS
- JavaScript
- PHP
- PWA manifest

## Running

Place the project on a PHP-enabled web server and open `index.html`.
